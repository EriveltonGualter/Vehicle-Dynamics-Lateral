Instruções

Para a utilização do pacote é necessário realizar os seguintes passos:

  • Baixar o pacote Dinamica-Veicular clicando em "Download ZIP" na página https://github.com/andresmendes/Dinamica-Veicular.
  • Salvar o pacote num path do MATLAB® ou adicionar o diretório na lista de paths. Mais detalhes em help Path.
  • Importar o pacote inteiro digitando no command window:
import DinamicaVeicular.*
