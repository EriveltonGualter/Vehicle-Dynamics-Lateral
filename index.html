Instructions

Installation steps:

  • Download the Vehicle Dynamics - Lateral repository by clicking here
  • Save the package (folder "+VehicleDynamicsLateral") in the MATLAB® path or add your current path to the paths list. More details in help path.
  • Import the entire package with the command:
import VehicleDynamicsLateral.*

Getting started

Run and explore the files TemplateSimple.m and TemplateArticulated.m.

Examples

The templates simulate vehicle systems according to the flowchart above.

  • Template Simple - Simple vehicle simulation.
  • Template Articulated - Articulated vehicle simulation.
  • Tire Comparison - Comparison of tire models.
  • Sinusoidal Steering - Maneuver with sinusoidal steering angle input.
  • Steering Control - Double lane change maneuver.
  • Kalman Filter - Kalman Filter application
  • Skid Pad - Simple vehicle moving in circle
